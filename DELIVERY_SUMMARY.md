### View the Gallery
1. Navigate to http://localhost:5173/ (dev server)
2. Click "About Us" tab
3. Scroll down to see the gallery
4. Try filtering, hovering, clicking images, using keyboard

### Customize Gallery
1. Edit `src/GalleryComponent.tsx`
2. Modify color values (search for #4CAF50)
3. Change animation timings (search for duration-)
4. Add/remove images from `galleryImages` array
5. Changes auto-reload in browser

### Add New Images
1. Place image in `/public/` folder
2. Add entry to `galleryImages` array
3. Follow existing format:
```typescript
{
  id: '15',
  src: '/new-image.jpg',
  alt: 'Descriptive alt text',
  category: 'facility' or 'doctor',
  title: 'Image Title',
  description: 'Brief description'
}
```
